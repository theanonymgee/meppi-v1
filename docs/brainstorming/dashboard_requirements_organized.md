# MEPPI Dashboard Requirements (정리된 버전)

## 개요 (Overview)

본 대시보드는 중동 및 북아프리카 18개국 스마트폰 시장의 가격 정보를 수집, 분석, 시각화하는 시스템입니다. 명목가격(Nominal Price)과 실질가격(Effective Price)을 비교하고, 통신사 요금제, 프로모션, 관세, 부가세 등 다양한 요소를 고려한 소비자 실구매가를 분석합니다.

## 핵심 기능 (Core Features)

### 1. 채널별 가격비교 (Channel Price Comparison)

**입력 파라미터:**
- 국가 선택
- 브랜드 선택
- 모델 선택

**출력 데이터:**

| 가격 유형 | 설명 |
|-----------|------|
| **1) 명목가격 (Nominal Price)** | 표시된 그대로의 가격 |
| **2) 실질가격 (Effective Price)** | 요금제, 스포트 프로모션 감안한 실구매가<br>• 마우스 호버 시 산정 로직 팝업 설명 |
| **3) 타국가 비교용 가격** | Effective Price에서 국가별 관세, 부가세 제외한 가격 |
| **4) 전국가 벤치마크 가격** | 아마존 UAE 최저가에 부가세 5% 제외한 가격 |

---

### 2. 경쟁사 비교 (Competitor Comparison)

**입력 파라미터:**
- 국가 선택
- 브랜드 선택
- 모델 선택

**경쟁 브랜드:**
- Samsung
- Apple
- Honor
- Xiaomi
- Infinix
- Tecno

**출력 데이터:**
1. **경쟁모델 선정**: 명목가격 기준 동일 가격대 모델 1개씩 리스트업
2. **스펙 비교**: 해당 모델의 사양 정보
3. **가격 비교**: 해당 국가의 채널 타입별 대표 채널 2개씩 선정
   - 명목가격
   - 실질가격 (요금제, 스포트 프로모션 감안)

---

### 3. 브랜드별 프로모션 조회 (Brand Promotion Timeline)

**입력 파라미터:**
- 특정 국가 선택
- 특정 브랜드 선택

**출력 데이터:**
- 선택 브랜드의 현재 판매 중인 **모든 모델**에 대해
- **주차별 타임라인**으로 표시
  - 명목가격
  - 실질가격
  - 프로모션 내용
  - 해당 모델 전용 특별 요금제

---

### 4. 중동 실질가격 비교 (Middle East Effective Price Comparison)

**입력 파라미터:**
- 브랜드 선택
- 모델 선택

**출력 데이터:**
- 전국가 해당 모델의 최저가 조사
- **3가지 가격 기준으로 비교:**
  1. 명목가격
  2. 실질가격 (Effective Price)
  3. 타국가 비교용 가격 (관세, 부가세 제외)
- 해당 채널 정보 함께 표시
- 전 국가 가격을 한눈에 비교

## 데이터 수집 전략 (Data Collection Strategy)

### 1. 모델 정의 (Model Definition)

**데이터 소스:**
- GSM-RAG 기반 스마트폰 정보 조사
- 해당 사이트에 없는 모델은 skip

### 2. 시장 조사 (Market Research)

**조사 대상 카테고리:**
- 통신사업자 (요금제 결합 소비자가격)
- 온라인 Pure Player
- 브랜드 공식 온라인 매장
- 전자기기 전문 리테일 온라인 사이트
- 하이퍼마켓 (까르푸 등)

**제외 기준:**
- 해당 카테고리가 전체 스마트폰 판매량 대비 비중이 너무 적으면 제외

### 3. 시장 데이터 파악 (Market Data Analysis)

**필수 조사 항목:**
1. **국가별 스마트폰 시장:**
   - 전체 스마트폰 사용자 수
   - 교체주기 감안한 연간 판매량
   - StatCounter 기반 vendor별 active 유저
   - 제3자 공신력 있는 외부 시장조사 데이터

2. **판매 채널 현황:**
   - 주요 통신사업자
   - 주요 전자기기 판매 체인
   - 온라인 Pure Player
   - 벤더 공식 온라인 매장
   - 각 채널별 스마트폰 매출 추정치

### 4. 가격 데이터 수집 (Price Data Collection)

**수집 대상:**
- 스마트폰 가격
- 요금제 정보
- 요금제 결합 실구매가
- 스포트성 프로모션
- 요금제 할인
- 번들 프로모션

**시스템 아키텍처:**
- 크롤링 시스템 구현
- 데이터베이스화 (DB)
- Cron 기반 매일 업데이트

### 5. 환율 정책 (Exchange Rate Policy)

**기준 통화:** 미국 달러 (USD)

**표기:**
- 달러 기준 국가별 비교
- 현지통화도 병행 표기

**특별 조사:**
- 공식환율보다 암시장 환이 더 지배적인 국가
- 암시장 환율 매일 조사 툴 구현
- 조사된 암시장 환율 적용

### 6. 벤치마크 가격 기준 (Benchmark Price Standard)

**중동 시장 특성:**
- 두바이 프리존(Free Zone) 홀세일링 공급 루트 다수

**벤치마크:**
- 아마존 UAE 최저가
- VAT 5% 제외한 가격

### 시스템 구성

**데이터 수집:**
- 웹 크롤링 아키텍처
- 자동화 스케줄링 (Cron)
- 일일 업데이트 시스템

**데이터베이스:**
- 모든 수집 데이터 DB화
- 가격 이력 관리
- 프로모션 이력 관리

**분석 로직:**
- 명목가격 vs 실질가격 비교
- 관세/부가세 계산 로직
- 요금제 결합 가격 계산
- 최저가 발견 알고리즘

### 핵심 기술 요구사항

1. **멀티 국가 지원**
   - 18개국 현지화
   - 다국어 지원
   - 현지 통화 표기

2. **실시간 데이터 업데이트**
   - 매일 자동 크롤링
   - 스포트 프로모션 감지
   - 변동 가격 추적

3. **정확한 가격 계산**
   - 요금제 할인 계산
   - 번들 프로모션 가치 산정
   - 관세/부가세 제외 로직
   - 암시장 환율 적용

4. **사용자 인터페이스**
   - 직관적인 필터링 (국가, 브랜드, 모델)
   - 마우스 호버 시 상세 설명
   - 타임라인 시각화
   - 비교 테이블

## 가격 정의 (Price Definitions)

### 명목가격 (Nominal Price)
- 광고나 표시된 그대로의 가격
- 할인, 요금제, 프로모션 미반영

### 실질가격 (Effective Price)
- 소비자가 실제 지불하는 가격
- **반영 요소:**
  - 통신사 요금제 할인
  - 스포트 프로모션
  - 번들 프로모션
  - 기타 혜택

### 타국가 비교용 가격
- 실질가격에서
- 관세 제외
- 부가세 제외
- 순수 제품 가격 비교

### 벤치마크 가격
- 아마존 UAE 최저가
- VAT 5% 제외
- 중동 지역 기준 가격
