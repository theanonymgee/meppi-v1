# MEPPI 대시보드 요구사항 정리

> 피터가 원했던 대시보드 스펙 (2026-02-11 기준)

---

## 🎯 핵심 기능 (4가지)

### 1. 채널별 가격비교 (Channel Price Comparison)
**입력:** 국가, 브랜드, 모델 선택

**출력:**
| 가격 유형 | 설명 |
|-----------|------|
| 명목가격 (Nominal Price) | 표시된 그대로의 가격 |
| 실질가격 (Effective Price) | 요금제, 프로모션 감안한 실구매가 (마우스 호버 시 산정 로직 팝업) |
| 타국가 비교용 가격 | Effective Price에서 관세, 부가세 제외 |
| 전국가 벤치마크 가격 | 아마존 UAE 최저가에 부가세 5% 제외 |

---

### 2. 경쟁사 비교 (Competitor Comparison)
**입력:** 국가, 브랜드, 모델 선택

**경쟁 브랜드:** Samsung, Apple, Honor, Xiaomi, Infinix, Tecno

**출력:**
- 명목가격 기준 동일 가격대 경쟁 모델 1개씩 리스트업
- 스펙 비교 테이블
- 채널 타입별 대표 채널 2개씩 가격 비교

---

### 3. 브랜드별 프로모션 타임라인 (Brand Promotion Timeline)
**입력:** 국가, 브랜드 선택

**출력:**
- 선택 브랜드의 **모든 모델**에 대해
- **주차별 타임라인**으로 표시
  - 명목가격
  - 실질가격
  - 프로모션 내용
  - 해당 모델 전용 특별 요금제

---

### 4. 중동 전국가 실질가격 비교 (Regional Price Comparison)
**입력:** 브랜드, 모델 선택

**출력:**
- 전국가 해당 모델 최저가 조사
- 3가지 가격 기준 비교 (명목, 실질, 타국가 비교용)
- 해당 채널 정보 함께 표시
